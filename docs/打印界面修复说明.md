# 打印界面修复说明

## 背景
本次仅针对麻醉记录单“打印界面”和对应网页实时单中与打印相关的时间轴显示问题进行修复，未改动其他业务模块。

## 已完成事项
1. 修复打印后错位与边线缺失问题  
   - 打印分区横线由“仅内部行线”改为“包含顶部与底部边界线”统一生成。  
   - 解决部分区域打印时少一条边线的问题。

2. 修复顶部时间刻度遮挡问题（网页显示）  
   - 移除时间栏背景重复线绘制，改为真实刻度线元素绘制。  
   - 提升时间文字层级并增加白底遮罩，避免竖线穿过时间文本。

3. 保持打印结构与 A4 输出一致性（既有修复延续）  
   - 打印网格继续采用真实线条元素（纵线/横线）而非背景图案。  
   - 打印时保留关键分区的行列对齐策略，减少药品、输液、生命体征区域错位风险。

4. 回归验证  
   - 单元测试通过：`npm.cmd test`（20/20 通过）。  
   - 构建通过：`npm.cmd run build`。
